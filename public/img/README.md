# Migration Step Images

This directory contains visual guides for each migration step.

## Expected Images

Place screenshot images here with the following naming convention:

- `step-1-add-zone.png` - Add Zone to Cloudflare
- `step-2-backup-config.png` - Backup Current Configuration
- `step-3-ssl-mode.png` - Choose SSL/TLS Mode
- `step-4-ssl-setup.png` - Set Up SSL/TLS Certificates
- `step-5-configure-dns.png` - Configure DNS Records
- `step-6-protect-origin.png` - Protect Origin Server
- `step-7-test-staging.png` - Test in Staging Environment
- `step-8-verify-ssl.png` - Verify SSL/TLS Configuration
- `step-9-convert-full.png` - Convert Records to Full Mode
- `step-10-enable-dnssec.png` - Enable DNSSEC
- `step-11-prepare-ns.png` - Prepare for Nameserver Change
- `step-12-change-nameservers.png` - Change Nameservers at Registrar
- `step-13-monitor.png` - Monitor and Optimize
- `step-14-iac-cicd.png` - Infrastructure as Code & CI/CD Pipelines

## Image Guidelines

- **Format**: PNG or JPEG
- **Size**: Recommended width 800-1200px
- **Content**: Clear screenshots from Cloudflare dashboard or relevant configuration screens
- **Quality**: High resolution, clearly readable text
- **Annotations**: Add arrows, highlights, or text to draw attention to important elements

The frontend will automatically display these images if they exist, or show a placeholder if missing.
